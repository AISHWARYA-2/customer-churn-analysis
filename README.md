# Customer Churn Analysis

## Overview
Analyzed telecom customer data to identify factors contributing to customer churn.

## Dataset
IBM Telco Customer Churn Dataset

## Technologies
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- Scikit-learn

## Key Insights
- Month-to-month customers had the highest churn rate.
- High monthly charges increased churn probability.
- Customer tenure was a strong retention indicator.

## Model Performance
Random Forest Accuracy: 82%
